Давайте почнемо! Виконуйте урок крок за кроком:

  1. Створіть папку з проектом, відкрийте проект за допомогою Visual Studio Code, створіть файл index.html та відкрійте його в браузері.

    Якщо ви ніколи такого не робили, то пройдіть спочатку цей урок .

  2. Вставте цей код між відкриваючим та закриваючим тегами body: 

    <select name="select" id="year-select">
  <option value="2000">2000</option>
  <option value="2001">2001</option>
  <option value="2002">2002</option>
  <option value="2003" selected>2003</option>
  <option value="2004">2004</option>
  <option value="2005">2005</option>
  <option value="2006">2006</option>
  <option value="2007">2007</option>
</select>

    Тепер ваш файл index.html має виглядати якось так: 

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <select name="select" id="year-select">
      <option value="2000">2000</option>
      <option value="2001">2001</option>
      <option value="2002">2002</option>
      <option value="2003" selected>2003</option>
      <option value="2004">2004</option>
      <option value="2005">2005</option>
      <option value="2006">2006</option>
      <option value="2007">2007</option>
    </select>
  </body>
</html>

  3. Збережіть файл та перезавантажте браузер на тій вкладці, де в вас відкрита index.html-сторінка.

    Ви маєте побачити в браузері елемент з вибором року. За замовчуванням активний рік має значення 2003. За це відповідає атрибут selected для елементу option.

  4. Тепер давайте додамо ще два елементи option в елемент для вибору року.

    Це будуть роки 2008 та 2009.

    І зробим ми це за допомогою JavaScript.

    Вставте цей код перед закриваючим тегом body: 

    <script>
  const yearSelectElement = document.getElementById('year-select');

  const optionElement = document.createElement('option');
  optionElement.setAttribute('value', 2008);
  optionElement.innerHTML = 2008;

  const optionElement2 = document.createElement('option');
  optionElement2.setAttribute('value', 2009);
  optionElement2.innerHTML = 2009;

  yearSelectElement.insertAdjacentElement('beforeEnd', optionElement);
  yearSelectElement.insertAdjacentElement('beforeEnd', optionElement2);
</script>

    Тепер ваш файл index.html має виглядати якось так: 

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <select name="select" id="year-select">
      <option value="2000">2000</option>
      <option value="2001">2001</option>
      <option value="2002">2002</option>
      <option value="2003" selected>2003</option>
      <option value="2004">2004</option>
      <option value="2005">2005</option>
      <option value="2006">2006</option>
      <option value="2007">2007</option>
    </select>

    <script>
      const yearSelectElement = document.getElementById('year-select');

      const optionElement = document.createElement('option');
      optionElement.setAttribute('value', 2008);
      optionElement.innerHTML = 2008;

      const optionElement2 = document.createElement('option');
      optionElement2.setAttribute('value', 2009);
      optionElement2.innerHTML = 2009;

      yearSelectElement.insertAdjacentElement('beforeEnd', optionElement);
      yearSelectElement.insertAdjacentElement('beforeEnd', optionElement2);
    </script>
  </body>
</html>

  5. Подивіться в браузері - тепер в селекті стало на два елемента більше.

  6. Тепер додамо ще більше елементів в наш селект за допомогою js.

    Вставте цей код перед закриваючим тегом script: 

    for (let year = 2010; year <= 2100; year++) {
          const optionElement = document.createElement('option');
          optionElement.setAttribute('value', year);
          optionElement.innerHTML = year;
        
          yearSelectElement.insertAdjacentElement('beforeEnd', optionElement);
        }

    Тепер ваш файл index.html має виглядати якось так: 

    <!DOCTYPE html>
        <html lang="en">
          <head>
            <meta charset="UTF-8" />
            <meta http-equiv="X-UA-Compatible" content="IE=edge" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Document</title>
          </head>
          <body>
            <select name="select" id="year-select">
              <option value="2000">2000</option>
              <option value="2001">2001</option>
              <option value="2002">2002</option>
              <option value="2003" selected>2003</option>
              <option value="2004">2004</option>
              <option value="2005">2005</option>
              <option value="2006">2006</option>
              <option value="2007">2007</option>
            </select>
        
            <script>
              const yearSelectElement = document.getElementById('year-select');
        
              const optionElement = document.createElement('option');
              optionElement.setAttribute('value', 2008);
              optionElement.innerHTML = 2008;
        
              const optionElement2 = document.createElement('option');
              optionElement2.setAttribute('value', 2009);
              optionElement2.innerHTML = 2009;
        
              yearSelectElement.insertAdjacentElement('beforeEnd', optionElement);
              yearSelectElement.insertAdjacentElement('beforeEnd', optionElement2);
        
              for (let year = 2010; year <= 2100; year++) {
                const optionElement = document.createElement('option');
                optionElement.setAttribute('value', year);
                optionElement.innerHTML = year;
        
                yearSelectElement.insertAdjacentElement('beforeEnd', optionElement);
              }
            </script>
          </body>
        </html>

Вітаю! Ви зробили елемент для вибору року та навчились створювати та додавати нові елементи за допомогою javascript.
